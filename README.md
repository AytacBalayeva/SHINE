## 📖 About

SHINE is a full-stack e-commerce application built to deepen my understanding of Next.js and modern web development practices. Through this project I explored server-side rendering, database management with Prisma and PostgreSQL, secure authentication with NextAuth, and real payment processing with Stripe — all the way from UI design to production deployment.

---

## ✨ Features

- 🔐 **User Authentication** via NextAuth
- 🛍️ **Shopping Cart** with quantity control
- 🔍 **Product Filtering** by category
- 💳 **Stripe Payment** integration
- 📱 **Responsive Design** across all devices

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/your-username/shine.git

# Install dependencies
cd shine
npm install

# Set up environment variables
cp .env.example .env

# Run the app
npm run dev
```
